# 15dirtyWords
Simple project to broaden your vocabulary and make sure you aren't using any of the 15 dirty words

First 15 words come from this article:
http://mashable.com/2015/05/03/words-eliminate-vocabulary/

Dirty Words:
* that
* went
* honestly
* absolutely
* very
* really
* amazing
* always
* never
* literally
* just
* maybe
* stuff
* things
* irregardless
